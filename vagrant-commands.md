
Vagrant Commands

The following are some popular/common vagrant commands. Note that doing a 'vagrant help' will also brings up command usage

```
vagrant init
```

```
vagrant box list
```

```
vagrant box add [box-name]
```

```
vagrant box remove [box-name]
```

```
vagrant up
```

```
vagrant status
```

```
vagrant suspend
```

```
vagrant resume
```

```
vagrant halt
```

```
vagrant destroy
```

```
vagrant provision
```

```
vagrant login
```

```
vagrant share
```

```
vagrant reload
```

```
vagrant help
```
